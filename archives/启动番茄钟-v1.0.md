# 启动番茄钟批处理文件 — v2

## 文件说明

`启动番茄钟.bat` — Windows 批处理文件，用于一键启动 Python HTTP 服务器并在浏览器中打开番茄钟应用。

## 版本历史

| 版本 | 日期 | 修改内容 |
|------|------|----------|
| v1 | 2026-06-16 | 初始版本：cd 到项目目录 → 启动 python http.server → 用 `start` 打开浏览器 |
| v2 | 2026-06-16 | 修复：添加 `chcp 65001` 避免中文乱码；`start` 改为 `start "" "url"` 兼容 Windows 的 start 命令参数解析 |

## v1 → v2 修改内容

### 问题
1. 使用 `Start-Process` 写入的文件因 PowerShell 默认编码（UTF-16 LE）导致批处理中的中文字符变成乱码
2. `start http://...` 在部分 Windows 环境中因缺少空标题参数 `""` 导致无法正确打开浏览器

### 修复
- 添加 `chcp 65001 >nul` 将代码页切换为 UTF-8，确保中文显示正常
- `start http://...` → `start "" "http://..."`，第一个 `""` 是窗口标题参数，防止 URL 被误解释为标题

### 当前内容
```batch
@echo off
chcp 65001 >nul
cd /d "F:\hermess\projects\20250611-番茄钟决策系统"
start /min python -m http.server 8080
timeout /t 2 /nobreak >nul
start "" "http://localhost:8080/pomodoro-v5.html"
echo.
echo 番茄钟服务器已启动！
echo 按任意键关闭服务器...
echo.
pause >nul
taskkill /f /im python.exe >nul 2>&1
echo 服务器已关闭。
timeout /t 2 /nobreak >nul
```
